# APPM Task Manager

A single-page task manager built with Vite and TypeScript. Add tasks, mark them complete, filter your list, and clear finished items — all persisted in your browser.

## Features

- Add, complete, and delete tasks
- Filter by All, Active, or Done
- Tasks saved to `localStorage` automatically
- Responsive, accessible UI

## Getting started

```bash
npm install
npm run dev
```

Open the URL shown in your terminal (usually `http://localhost:5173`).

## Scripts

| Command         | Description              |
| --------------- | ------------------------ |
| `npm run dev`   | Start development server |
| `npm run build` | Production build         |
| `npm run preview` | Preview production build |

## Deploy

Run `npm run build` and deploy the `dist/` folder to any static host (GitHub Pages, Netlify, Vercel, etc.).
